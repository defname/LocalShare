Help & Documentation

LocalShare makes it easy to share files and content directly over your local network. The best part: No internet access is required. Your data stays private, entirely offline, and moves at the maximum speed of your Wi-Fi.

Recipients don't need to install any app. LocalShare is compatible with virtually every device and operating system (Windows, iOS, Linux, Android, etc.) – all you need is a standard web browser on the client side.

JavaScript Requirement: Currently, a web browser with JavaScript enabled is required for the interface to function. We are working on a non-JS fallback for future updates to ensure even broader compatibility.

Selecting Files and Content

LocalShare provides a flexible way to make your data available to other devices on your network.

Sharing Files

You can add files by tapping the Plus (+) button within the app or by using the standard Android Share Sheet from any other application (e.g., Photos, Gallery, or File Manager).

Shared files can be organized directly in the Main Screen or in the Files Screen accesible via the Menu.

Sharing Text

To share text, simply highlight it in any app and select LocalShare in the Android share-sheet. You can also paste your current clipboard content directly as a new entry within the app.

These items are organized in the Shared Content screen.

Start Sharing

Once you have selected your files or content, tap the Start Server button to make them available to your local network.

When the server is running, all shared items are accessible via a clean web interface. Recipients can connect by scanning the QR Code or opening the provided URL in their browser.

ZIP Download

Recipients can download all shared files at once as a single ZIP archive. This feature works even without JavaScript.

Easy Access

Share the link via any messaging app or let others scan the QR code for instant connection.

Dynamic Updates: You can add more files or content while the server is active. New items appear instantly in the web interface for everyone.

Pro Tip: No Wi-Fi available? Simply start your phone's Mobile Hotspot and let the other device connect to it. LocalShare works perfectly over your own hotspot, making it ideal for sharing files in the car, on a plane, or outdoors.

Common Use Cases

Quick Phone-to-PC Transfer

Send photos, documents or other files from your phone to your computer instantly without hunting for a USB cable or using cloud services.

Local Video Streaming

Stream movies or recorded videos from your phone to any tablet, PC, or Smart TV browser with full seeking support.

Digital Photo Slideshow

Show off your latest vacation pictures on a bigger screen using the integrated slideshow feature in the web interface.

Multi-Device Distribution

Share files or notes with a whole group simultaneously (e.g., in a meeting or classroom). Everyone just scans the same QR code.

Cross-Device Clipboard

Copy long URLs, Wi-Fi keys, or snippets on your phone and access them on your laptop via the Shared Content section.

True Offline Mobility

Perfect for traveling, planes, or areas with no signal. Your phone becomes a local server for everyone around you.

Security & Privacy

LocalShare is built to be secure and private by default:

No Cloud Storage

Your files never touch the internet. Transfers happen directly between devices in your local network.

Access Tokens

Access is protected by a cryptographic token. Only users with the link or QR code can access the content. You can manually refresh or change this token in the settings.

Manual Approval

When enabled, you must manually allow new devices to connect via a notification on your phone.

Technical Note: Local Network Security

Files are transferred via HTTP (unencrypted). Modern browsers do not accept security certificates for local IP addresses without displaying severe "unsafe connection" warnings, which would break the user experience.

Safety Recommendation: Only use LocalShare in trusted, private Wi-Fi networks. Avoid sharing sensitive data on public or untrusted hotspots.

Frequently Asked Questions (FAQ)

Why can't my device find the server?

Ensure both devices are on the same Wi-Fi network. Check if "AP Isolation" or "Guest Network" settings on your router are blocking communication between local devices.

Is there a file size limit?

No. Since the transfer happens locally, it's only limited by your available storage and network speed.
